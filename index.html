As we are developers , we need a text editor to write the codes.
there are many text editors (free, paid ,come with OS, need to be downloaded ) with deferent features.
So we need to determine what suite us .
I think the best one should includes the :

  1. code completion(autocomplete ,suggestion and closing of tags).
  2. syntax highlighting(make the code more noticeable by colorizing the text)
  3. themes (to reduce eye strain and fatigue like dark themes)
  4. the ability add extensions to make the work more efficiently.
Most popular text editors is:
  • Visual Studio Code
  • NotePad++
  • Atom
  • Brackets
  • Sublime Text
There is another software called IDE (Integrated Development Environment) . It is software package comes together all in one ( text editor, a file manager, a compiler, and a debugger) like Visual Studio and Android Studio.
